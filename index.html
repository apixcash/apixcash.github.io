What APIXCASH?

APIXCASH (APX) project launched on 22th, Sept 2022, when the genesis block got mined, We are an open-source project designed to enable instant payments to ayone around the world.

Truly decentralized.

Apixcash has never been funded through an ICO or any premine as Apixcash never had a premine. There is no CEO or a company controlling the Apixcash blockchain. It is a volunteer based and global community driven project.

More secure.

Apixcash uses the cryptographic algorithm called scrypt to allow bigger miners called ASICs to further secure the network from attacks.

Much faster.

Apixcash blocks are mined every 1 minutes which is 4 times faster than Bitcoin. SegWit implementation and blockchain rigidity enable up to 1440 on-chain transactions per second with negligible fees.

Developer Friendly.

Our developer is Friendly to community, this is an important point for us for the progress of the Apixcash

This is how the Apixcash blockchain works.

The three layers are the most innovative parts of the Apixcash blockchain providing the network infrastructure, security and communications to function with cutting edge speed.

Applications / Software.

The top layer is like full of different things like apps and software that all use Apixcash. contracts can be created on the Apixcash Core Client and many other applications can use the Apixcash Core to accept payments in milliseconds.

Digital asset / Public ledger.

The middle layer provides security and administration. A digital byte of data, a representation of larger data or a unit that holds value, and cannot be counterfeited, duplicated or hacked. An immutable public ledger where all transactions on the Apixcash network are recorded. Apixcash uses the scrypt algorithm for security. Newly minted Apixcash only come from mining.

Core protocol / Global network.

The bottom layer provides communication and operating procedures. A very low-level way nodes on the Apixcash global network communicate. Hundreds of people are running many versions of the Apixcash software all across the planet. Any server, computer, tablet or mobile phone connected to the Apixcash network becomes a node that helps relay transactions.

Open source / Permissionless.

The Apixcash blockchain is completely open source and free to use released under the MIT license, which gives you the power to run and modify the software. Transparency allows for independent verification of binaries and their corresponding source code.

A digital currency

Apixcash (APX) is a highly scalable peer-to-peer digital currency, blockchain economy that enables a whole new world of transacting with low fees.

Easy to use.

Getting started to use Apixcash is easy. All you need to do is install a Apixcash supported wallet then install it to send and recieve Apixcash.

Not an ICO.

Apixcash has never been funded through an ICO or any premine as Apixcash never had a premine. There is no CEO or a company controlling the Apixcash blockchain. It is a volunteer based and global community driven project. The founder, developers and the community are unpaid volunteers that eliminate the risk of bankrupt and corruption.

Much faster.

Apixcash transactions are confirmed in about 1 minutes, much faster than Bitcoin. SegWit enables up to 1400 on-chain transactions per second and negligible fees make Apixcash another ideal currency for daily payments.

Always secure.

Apixcash has been built on the uncrackable technology of the Bitcoin and Litecoin blockchain and implemented various advanced enhancements over it such as the Lightning Network.

Mineable.

Apixcash is a 100% Proof of Work (PoW) blockchain that can be mined with the scrypt algorithm.

Manageable units.

Compared to 21 Billion RVN, there will be 50 million Apixcash. The Apixcash block reward decreases 50% every 1.000.000 blocks.
